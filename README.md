# Color Palette Generator

A GUI application for generating color palettes from a base color.

## Requirements

- Python 3.6 or higher
- No external libraries required (uses Python standard library)

## How to Run

```bash
python color_palette_generator.py
```

## How to Use

1. Launch the application
2. Enter a color (hex code like `#3498db` or color name like `blue`)
3. Click Generate or press Enter
4. Click any color to copy it to clipboard
